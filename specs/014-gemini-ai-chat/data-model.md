# Data Model: Gemini AI Chat Integration

## New Entities

### AIConfig (runtime configuration, loaded from env)

| Field | Type | Description |
|-------|------|-------------|
| `apiKeys` | `string[]` | List of Gemini API keys, loaded from `GEMINI_API_KEYS` env var |
| `models` | `string[]` | List of Gemini model names, loaded from `GEMINI_MODELS` env var |
| `currentKeyIndex` | `number` | Current API key index for rotation |
| `currentModelIndex` | `number` | Current model index for rotation |
| `keyStates` | `KeyState[]` | Runtime state per key (failure tracking) |
| `requestTimeoutMs` | `number` | Timeout per AI request (default: 30000) |
| `moderationTimeoutMs` | `number` | Timeout for moderation requests (default: 5000) |
| `suggestionTimeoutMs` | `number` | Timeout for suggestion requests (default: 10000) |
| `maxContextMessages` | `number` | Max messages to send as context (default: 10) |

```typescript
interface KeyState {
  apiKey: string
  consecutiveFailures: number
  lastErrorAt?: Date
  isRateLimited: boolean
}
```

---

### AIUserSettings (persisted in MongoDB)

| Field | Type | Description |
|-------|------|-------------|
| `_id` | `ObjectId` | Primary key |
| `userId` | `ObjectId` | Reference to User |
| `chatbotEnabled` | `boolean` | AI chatbot feature toggle (default: true) |
| `suggestionsEnabled` | `boolean` | Smart reply suggestions toggle (default: true) |
| `moderationEnabled` | `boolean` | Content moderation toggle (default: true) |
| `createdAt` | `Date` | Creation timestamp |
| `updatedAt` | `Date` | Last update timestamp |

**Relationships**:
- One-to-One with User (`userId`)

---

### ModerationResult (runtime, not persisted)

| Field | Type | Description |
|-------|------|-------------|
| `messageId` | `string` | ID of the analyzed message |
| `sentiment` | `"positive" \| "neutral" \| "negative"` | Detected sentiment |
| `sentimentScore` | `number` | Score from -1.0 (negative) to 1.0 (positive) |
| `toxicityScore` | `number` | 0.0 to 1.0 (higher = more toxic) |
| `isToxic` | `boolean` | True if toxicity threshold exceeded |
| `warningMessage` | `string?` | Optional warning text for UI display |

---

### AISuggestion (runtime, not persisted)

| Field | Type | Description |
|-------|------|-------------|
| `conversationId` | `string` | ID of the conversation |
| `suggestions` | `string[3]` | Exactly 3 reply suggestions |
| `generatedAt` | `Date` | Timestamp when generated |

---

### AIConversationContext (runtime, ephemeral)

| Field | Type | Description |
|-------|------|-------------|
| `conversationId` | `string` | ID of the conversation |
| `messages` | `ChatMessage[]` | Last N messages for context |
| `lastUpdated` | `Date` | When context was last updated |

```typescript
interface ChatMessage {
  role: 'user' | 'model'
  text: string
  timestamp: Date
}
```

| `isAIBotMessage` | `boolean` | True if this message was generated by AI bot (default: false) |

## Schema Summary

| Entity | Storage | Persistence |
|--------|---------|-------------|
| `AIConfig` | Memory/env | Ephemeral (runtime) |
| `AIUserSettings` | MongoDB | Persisted per user |
| `ModerationResult` | Memory | Ephemeral per message |
| `AISuggestion` | Memory | Ephemeral per conversation |
| `AIConversationContext` | Memory | Ephemeral, cached |
| `Message` (extended) | MongoDB | Persisted with new fields |

## State Transitions

### Key Rotation State Machine

```
READY ──(429/403)──> RATE_LIMITED
RATE_LIMITED ──(wait 60s)──> READY
RATE_LIMITED ──(3 consecutive failures)──> SKIPPED
SKIPPED ──(rotate key)──> READY
```

### AI Feature Toggle

```
ENABLED <──> DISABLED (user-controlled, persisted)
```
